# Advanced Face Recognition Attendance System

## Overview
This project is an AI-based attendance management system that uses face recognition technology to automatically detect and mark student attendance.

## Features
- Face Detection
- Face Recognition
- Automatic Attendance Marking
- Student Registration
- Attendance Reports
- CSV File Export
- Real-Time Camera Detection

## Technologies Used
- Python
- OpenCV
- Tkinter
- NumPy
- Pandas

## Project Structure
- train.py → Train face data
- attendance.py → Attendance system
- login.py → Login page
- signup.py → Registration system

## Installation
1. Clone repository
2. Install dependencies
3. Run main.py
